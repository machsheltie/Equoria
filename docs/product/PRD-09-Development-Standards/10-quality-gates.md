# 10. Quality Gates

### 10.1 Pre-Commit

- [ ] ESLint passing
- [ ] TypeScript type-check passing
- [ ] Affected tests passing

### 10.2 Pre-Push

- [ ] Full test suite passing
- [ ] Build successful
- [ ] No console.log statements

### 10.3 Pre-Merge

- [ ] Code review approved
- [ ] All CI checks passing
- [ ] Documentation updated
- [ ] No breaking changes (or documented)
